# Eddy Current Probe Sensitivity Case Study

*Analysis of a 23 Full Factorial Design*
This case study demonstrates the analysis of a 23 full
factorial design.

The analysis for this case study is based on the
[EDA approach](../advanced-topics/eda-approach.md) discussed in an
earlier section.

*Contents*
The case study is divided into the following sections:

- Background and data
- Initial plots/main effects
- Interaction effects
- Main and interaction effects: block
       plots
- Estimate main and interaction effects
- Modeling and prediction equations
- Intermediate conclusions
- Important factors and parsimonious
       prediction
- Validate the fitted model
- Using the model
- Conclusions and next step
- Work this example yourself
